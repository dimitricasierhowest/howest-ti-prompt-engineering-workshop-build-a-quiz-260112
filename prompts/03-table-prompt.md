prompt:




Ontwerp en implementeer een **Single Page Application (SPA)** voor het **periodiek systeem der chemische elementen** met uitsluitend **HTML, CSS en vanilla JavaScript**.

**Technische eisen**

* Gebruik exact **één `index.html`**
* Alle **HTML, CSS en JavaScript volledig inline**
* Kopieer **alle data uit `atomen.json` inline** in een `<script>` (geen externe bestanden of fetch)

**Functionele eisen**

* Toon het **klassieke Mendelejev-periodiek systeem** in tabelvorm
* Implementeer een **zoekfunctie** die filtert op:

  * naam
  * symbool
  * atoomnummer
  * soort
  * toestand
* Bij **klik op een element** verschijnt een **detailweergave binnen dezelfde pagina** met **alle beschikbare gegevens** van dat element

**Vormgeving**

* Gebruik de **visuele stijl van de IMDb-website** (zoals in de meegeleverde screenshot): donkere achtergrond, duidelijke typografie, kaart-/paneelachtige lay-out, moderne UI
* Geen externe libraries of frameworks

**Resultaat**

* Eén zelfstandig draaiend HTML-bestand dat direct in de browser werkt
* Overzichtelijk, performant en gebruiksvriendelijk
